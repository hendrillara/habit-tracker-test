# Design (repo-only)

This folder contains the product’s design package in Markdown (no Figma required).

Create/refresh these artifacts for greenfield, user-facing work:

- `docs/ias/design/brief.md`
- `docs/ias/design/system.md`
- `docs/ias/design/flows.md`
- `docs/ias/design/screens.md`

Use `./scripts/ias init-design` to scaffold from templates.

Screenshots (optional) can live under `docs/ias/design/assets/` for local review.
