# Intuition loop (canonical)

This is the canonical “Niklas-style” loop agents should follow.

## Loop

1) Anchor the base goal (outcome statement).
2) Build the world model (environment + stakeholders + incentives).
3) Extract 100% true constraints (hard constraints).
4) Define success in value terms (acceptance criteria + proxy metrics).
5) Choose an execution mode (quality vs speed is a consequence).
6) Decompose into subgoals + required artifacts.
7) Execute and iterate, repeatedly re-anchoring on goal + hard constraints.

## Required persona passes (for non-trivial work)

- User pass: “how does this feel; is it obvious; is it worth it?”
- PM/value pass: “does this achieve the goal; is scope right?”
- Research pass (if needed): “what external facts/standards do we need; what are the unknowns?”
- Implementation pass: “is it maintainable; what are the risks?”
- Review pass: “what breaks; what’s unclear; what’s missing?”
